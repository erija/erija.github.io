Como funciona

Veja como é simples de usar em 3 passos.

1 Preparar...

Exporte os dados dos contra-cheques.

Use seu sistema de folha de pagamento para isso. Esses sistemas possuem uma funcionalidade de exportação de dados que inclusive já deve ser usado hoje para enviar os dados de contra-cheque para impressão dos seus contra-cheques.

2 Apontar...

Faça o upload dos dados exportados.

No HOLETRITE você escolhe se o arquivo enviado é referente a pagamentos mensais ou declaração de rendimentos para o IRPF. Apoś o envio, os dados serão processados para garantir que não ocorram inconsistências. Tudo automático.

3 Já!

Os dados estão disponíveis para consulta.

Pronto! Seus colaboradores já podem consultar seus contra-cheques a partir de diferentes dispositivos computadores, smartphones ou tablets.
